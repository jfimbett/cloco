## Lit Review Scoring Patterns

- BibTeX entries citing the WRONG paper (not just missing fields) warrant -15. This is worse than missing entries.
- Missing foundational methods papers (BLP for structural, Rosen for hedonic) are -10 to -15 each when the paper explicitly uses those methods.
- A structural project type with zero structural estimation methods papers is a -15 deduction.
- A paper mentioned in the frontier map text but absent from the annotated bibliography and BibTeX is a -5 (e.g., GPV 2000).
- Journal misattributions in the frontier map are -2 each.
- Proximity score miscalibrations are -1 each unless they affect the paper's positioning argument.
- Working paper reliance: 1/45 is fine. Flag only if >50% or if the unpublished paper is load-bearing for identification.
- The 80-point threshold is appropriate: a review can pass with residual low-severity gaps as long as no methods foundation is missing.

**Why:** Prevents score inflation on re-reviews. The jump from 62 to 82 reflected genuine fixes (BibTeX corrected, BLP/Nevo added, frontier map produced), not generosity.

**How to apply:** On re-reviews, verify each prior Required Action was actually completed before removing the deduction. Do not assume fixes were made.
