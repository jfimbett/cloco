# Academic Editor -- Agent Memory

## Scoring Calibration

- A 42-paper bibliography scored 68/100 when it had placeholder rows instead of actual citations for a claimed key contribution. Lesson: placeholder references are treated as missing.
- Cheap-talk experimental literature commonly omitted when only Crawford-Sobel (1982) is present. Always check for experimental follow-ups.
- oTree/z-Tree platform citations frequently forgotten in experimental economics lit reviews.

## Common Gaps by Topic

- **Deskilling / skill atrophy**: Key papers = Parasuraman-Manzey (2010), Dell'Acqua et al. (2023), Autor (2015), Bainbridge (1983).
- **CDA microstructure foundations**: Gode-Sunder (1993, JPE) often missing from CDA experiment lit reviews.
- **Task-based automation**: Acemoglu-Restrepo (2019, 2022) needed when AI autonomy levels are a treatment variable.
